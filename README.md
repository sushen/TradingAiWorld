# Trading AI Landing Page

Static, mobile-first landing page for **Trading AI** built with:

- HTML
- CSS
- JavaScript

## Project Structure

```text
project/
  index.html
  assets/
    css/
      style.css
    js/
      main.js
    images/
  downloads/
    trading-ai.zip
```

## GitHub Pages Deployment

1. Create a GitHub repository.
2. Upload project files.
3. Go to **Settings -> Pages**.
4. Select branch **main** and publish the site.
